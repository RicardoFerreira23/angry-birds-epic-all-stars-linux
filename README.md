echo "# Angry Birds Epic All Stars Linux Launcher

This launcher allows you to run the All Stars version of Angry Birds Epic on Linux seamlessly. 
It copies the Windows game files into a Linux-native directory and runs them via Wine or Proton.

## Usage

1. Run the launcher with:

\`\`\`bash
./launcher/run.sh
\`\`\`

2. The first time, select your All Stars game folder.
3. The launcher will copy the files and remove the originals.
4. Next time, just run the launcher.
